# 🔧 MAC-CHA (MAC Address Changer)

> A simple Python tool to temporarily change MAC addresses on Linux systems.

## 📌 About / Haqqında

**EN:**  
MAC-CHA is a lightweight Python tool that allows you to change your network interface MAC address temporarily.  
Designed for **educational and cybersecurity practice purposes**.

**AZ:**  
MAC-CHA Linux sistemlərində şəbəkə interfeysinin MAC ünvanını müvəqqəti dəyişmək üçün hazırlanmış sadə Python alətidir.  
**Tədris və kiber təhlükəsizlik məşqləri üçün nəzərdə tutulub.**

## ⚙️ Features / Xüsusiyyətlər

- 🔹 Simple and easy to use  
- 🔹 Works on Linux systems  
- 🔹 Supports manual MAC input  
- 🔹 Lightweight (no extra dependencies)
